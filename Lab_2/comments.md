Стандарт языка: C++11 (используются std::vector, исключения, строгая типизация, RAII в тестах)

ОС: Windows

API: WinAPI (CreateThread, WaitForSingleObject, CloseHandle, Sleep)

Проект состоит из следующих компонентов:
creating_threads.cpp — основная консольная программа;
creating_threads.h — заголовочный файл с объявлением потоковых функций и общих данных;
test.cpp — модуль интеграционного тестирования (Google Test).


Для проверки корректности работы потоков реализованы интеграционные тесты с использованием Google Test.

Проверяемые сценарии:

корректное вычисление минимума, максимума и среднего арифметического;

массив с одинаковыми элементами;

массив из одного элемента;

корректная модификация массива после завершения потоков.

В проекте реализована базовая обработка ошибок:
проверка корректности ввода размера массива;
обработка ошибок создания потоков (CreateThread);
корректное освобождение ресурсов при ошибках;
защита от некорректных входных данных (нулевая или отрицательная размерность массива).
