Корректно и полностью указывать путь к файлу.

Проект представляет собой консольное приложение на C++, состоящее из трёх взаимосвязанных программ,
демонстрирующих работу с бинарными и текстовыми файлами и межпроцессное взаимодействие через запуск дочерних процессов.

Стандарт языка: C++11 (используются std::vector, std::wstring, to_wstring, исключения)
ОС: Windows
API: WinAPI (CreateProcessW, STARTUPINFO, PROCESS_INFORMATION)

Ввод/вывод:
iostream — консольный ввод/вывод
fstream — работа с файлами

Типы данных: структуры (struct), все программы используют общую структуру данных employee.
Кодировка: Unicode (UTF-16), используется wmain, wstring, CreateProcessW.

Архитектура проекта
Проект состоит из трёх отдельных исполняемых файлов:
Creator — создание бинарного файла с данными сотрудников
Reporter — формирование текстового отчёта
creating_processes — управляющая программа, запускающая Creator и Reporter в качетсве дочерних процессов

В проекте реализована обработка ошибок:
проверка корректности числового ввода;
обработка ошибок открытия файлов;
обработка ошибок создания процессов;
защита от некорректных параметров (отрицательные значения, нулевое количество записей).
Используются стандартные исключения (std::invalid_argument, std::out_of_range)
